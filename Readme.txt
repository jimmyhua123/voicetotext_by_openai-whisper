### **語音轉文字工具使用說明**

#### **功能概述**
本工具提供簡單易用的圖形化界面，將音訊檔案（如 MP3、WAV、FLAC）轉錄為文字檔案，並可選擇是否包含時間碼。

---

#### **工具特性**
1. **支援多種音訊格式**：MP3、WAV、FLAC、M4A。
2. **逐字稿生成**：將音訊檔案轉換為文字並儲存為文字檔案。
3. **時間碼選項**：可選擇在輸出的文字檔案中包含時間碼。
4. **動態進度顯示**：在界面上實時顯示轉錄進度。
5. **一鍵安裝依賴**：首次使用時可自動安裝所需套件。

---

#### **安裝依賴**
如果是首次使用，請先點擊「**安裝依賴**」按鈕，系統會自動安裝以下必要套件：
- **`openai-whisper`**：用於語音轉文字。
- **`ffmpeg-python`**：處理音訊檔案格式。

---

#### **使用步驟**
1. **啟動工具**：
   雙擊啟動程式，界面將顯示主要功能。

2. **安裝依賴**（首次使用時執行）：
   - 點擊「安裝依賴」按鈕。
   - 等待提示「所有依賴已安裝完成！」。

3. **選擇音訊檔案**：
   - 點擊「選擇檔案」按鈕。
   - 從檔案選擇器中選擇您需要轉錄的音訊檔案。

4. **選擇時間碼選項**：
   - 勾選「包含時間碼」複選框，輸出檔案中將包含每段語音的時間碼。

5. **開始轉錄**：
   - 程式將自動處理音訊並生成逐字稿。
   - 逐字稿會儲存為與音訊檔案相同目錄的文字檔案，檔名格式為：
     ```
     <原音訊檔名>_transcription.txt
     ```

6. **檢視結果**：
   - 轉錄完成後，程式將提示逐字稿儲存位置。
   - 打開文字檔案檢視內容。

---

#### **注意事項**
1. **依賴安裝**：
   若安裝依賴過程中出現錯誤，請檢查網路連線或手動執行以下命令：
   ```bash
   pip install openai-whisper ffmpeg-python
   ```

2. **音訊檔案格式**：
   本工具支援常見音訊格式，但需確保音訊檔案清晰且不過長，否則可能影響轉錄結果。

3. **硬體需求**：
   - **CPU 模式**：適合處理一般音訊檔案。
   - **GPU 模式**（如有 CUDA 支援）：提升處理速度。
